# Модуль 2. Заняття 3. Селектори та колір.

1. Що таке css;
2. Підключення стилів (inline, embeded, external);
3. Селектори:
   - універсальний селектор;
   - селектор елемента;
   - селектор id;
   - селектор класу;
   - селектор нащадка;
   - селектор дитини;
4. Каскад стилів, специфічність, наслідування;
5. Псевдокласи стану (:hover, :focus, :active);
